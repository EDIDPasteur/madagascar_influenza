# Madagascar Influenza

## Goal

This repository aims to characterize avian influenza sequences from Madagascar and Africa available in GISAID, by addressing the following questions:

- Which avian influenza sequences exist from Madagascar and Africa in GISAID?
- What is their geographic location, sampling time, and whether the genomes are complete or not?

## Resources

- **GISAIDR** — R package for programmatic access to GISAID: https://github.com/Wytamma/GISAIDR
- **Nextstrain H5N1 HA (2y)** — Phylogenetic overview of recent H5N1 sequences: https://nextstrain.org/avian-flu/h5n1/ha/2y
